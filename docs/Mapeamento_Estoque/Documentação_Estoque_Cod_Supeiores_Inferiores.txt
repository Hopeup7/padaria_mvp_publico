📌 Objetivo deste Checkpoint
Documentar a estrutura padronizada de cadastro e controle de insumos adotada no MVP, estruturada em blocos hierárquicos por categoria, produto, marca e variação (tipo). Esta estrutura será utilizada para:
- Controle detalhado de estoque (volume, validade, valor unitário)
- Mapeamento direto com o cardápio (via Código_Sup)
- Cálculo de custo de produção e baixa automática
- Integração futura com banco de dados SQLite e painéis de gestão no Flask

🧱 Estrutura Oficial Definida
estoque = {
    "Categoria": {  # Ex: "Bebidas", "Alimentos", "Hortifruti", "Utensílios", etc.
        "Subcategoria": [  # Ex: "Águas", "Sucos", "Cervejas", etc.
            {
                "Código": "string",      # Código Superior do Produto, Ex: "Ag001"
                "Marca": "string",       # Nome completo do produto
                "Tipos": [               # Lista de variações (Tipo de embalagem, volume, vencimento, preço, etc.)
                    {
                        "Código_Tipo": "string",       # Código Inferior Ex: "TAg001.1"
                        "Tipo_Embalagem": "string",    # Ex: "Garrafa", "Pacote", "Lata"
                        "Volume": float or int,        # Ex: 600
                        "Unidade": "string",           # Ex: "ml", "g", "un", "kg"
                        "Quantidade_Total": int,       # Quantidade disponível
                        "Preço_Unitário": float,       # Valor por unidade
                        "Vencimento": "DD/MM/AAAA" or None
                    }
                ]
            }
        ]
    }
}

📌 Padronização de Códigos Superiores e Inferiores
Este documento define a estrutura padronizada dos sub-estoques do MVP da padaria, garantindo rastreabilidade eficiente e organização clara dos produtos.
Cada categoria segue a lógica de códigos superiores (produtos principais) e códigos inferiores (variações).

🥩 Carnes
- Código Superior: CXXX → Produto principal (ex: C001 = Picanha Maturatta - Friboi)
- Código Inferior: TCXXX → Embalagens e conservação (ex: TC001.1 = Picanha - Pacote Resfriado)


🌾 Grãos
- Código Superior: GXXX → Produto principal (ex: G001 = Arroz Camil Tipo 1)
- Código Inferior: TGXXX → Embalagem e volume (ex: TG001.1 = Arroz - Pacote 1kg)


🏺 Condimentos
- Código Superior: CdXXX → Produto principal (ex: Cd009 = Óleo de Soja - Liza)
- Código Inferior: TCdXXX → Variações de embalagem/volume (ex: TCd009.1 = Liza - Latão 18L)
